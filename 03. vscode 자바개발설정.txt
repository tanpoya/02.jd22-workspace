[ vscode 자바개발환경 설정 ]
1. 먼저 JDK를 설치후 환경변수 설정
2. vscode에 Java 확장 패키지를 설치
- 마켓플레이스에서 "Java"라고 검색
Extension Pack for Java 설치

3. 자바 프로젝트를 실행
1) F1키 누름 : "Java project"
-> "Create Java Project..." 클릭
2) "No build tools" 선택
3) 상위지정폴더 선택(생성)
4) 프로젝트 명을 쓰면 새로운 자바프로젝트가
vscode 새창으로 열림

4. 자바파일 생성 후 기본 실행하기
- 기본자바파이은 내부 클래스명과 동일한 파일명을 쓴다
- 클래스는 반드시 대문자로 시작한다
- 따라서 파일명도 대문자로 클래스명과 동일하게 작성함
